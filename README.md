# Lab-Extra--Tic-Tac-Toe

Laboratorio desarrollado siguiendo el tutorial oficial de React: [Tutorial: Tic-Tac-Toe](https://react.dev/learn/tutorial-tic-tac-toe#what-are-you-building).
Se agregó estilo personalizado usando CSS para mejorar la apariencia visual del juego.

## Video explicación

